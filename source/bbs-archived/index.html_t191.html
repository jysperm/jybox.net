接下来给大家介绍一个非常不错的网盘，Goolge Code
与其他网盘相比有如下优点:
空间大：2GB
限制少：单文件100MB，不限文件格式
下载速度快
可外链文件
因为Google Code目前还是英文版，因此在这里再介绍一下申请方法。
首先，用浏览器打开Goole Code创建项目的网页
http://code.google.com/hosting/createProject
如下图：

[attachment=61]
哦！全是英文，别着急，点击Create an account now，创建一个用户，如下图

[attachment=62]
下面填写注册表单，下图对英文进行了注释，请按照下图填写

[attachment=63]

值得注意的是，谷歌的验证码不易填写，请耐心多试几次
接下来是一个提示页面，大意是请到邮箱中查看邮件，我就不上图了
然后去邮箱中找邮件，邮件内有两个链接，注意！请一定点击第一个链接！

点开链接后是一个提示页面，大意是已经注册成功
我们再回到Goole Code创建项目的网页
http://code.google.com/hosting/createProject
接下来 来创建项目(也就是网盘空间)
进入这个页面后有如下表单：
[attachment=64]

上面对英文进行了注释，请按照上图填写
注册成功会看见如下页面：
[attachment=65]

点击"Download"
[attachment=66]

再点击"new download"，上传文件
按照下文的提示上传文件
[attachment=67]

上传后，你就可以在文件列表中看到你的文件了
[attachment=68]

点击进去，就可以下载文件
在下载按钮上点右键>属性就能看到直链地址

[attachment=69]
