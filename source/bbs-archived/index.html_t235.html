Android刷机教程简介

对于Android用户来说，干的最频繁的事除了上上网、聊聊天、偷偷菜、看看片、玩玩游戏以外，就是刷机了……我不是胡说，Android有的时候确实挺烦人的……但是，不会刷机的人肯定也是有的……
注：本次介绍的不会很全，因为缺少需要的东西，下次会补全的！
先介绍一下什么是刷机：
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
刷机就是一种改变你手机操作系统的一种行为，相当于给电脑装上不同版本的windows或者电脑重装系统，也就是刷手机的操作系统吧。目前常说的刷机就是用“更改替换了一些图片、铃声或菜单后的软件版本”来替换“手机中原有版本”的过程。大多数厂家的手机都是可以刷机的。
……………………………………………………………………………………………………………………………………
首先，任何品牌机都一样，只要还有可观的销售额，还在生产销售，官方会对同一部手机的内部软件系统升级，从而解决前版的一些缺陷来完善手机从而增加销售量，所以各大品牌机都可升级与刷机，但在工具与操作方面大不相同，效果更是不同。
……………………………………………………………………………………………………………………………………
你可以从官网下载，也可以从PC套件里点击软件更新，它会引导你下载并安装NSU，在线升级只是官方为方便用户实时的从官方网站更新手机内用软件，让用户可以用到官方推出的新版本，此方法简单，会保留你手机内的用户数据，风险低，需网络要求。
……………………………………………………………………………………………………………………………………
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
注意事项：
　　1.只要是和电脑有连接的机器就能自己刷，不论是正常使用中还是白屏中。如果手机已经与电脑无连接反映（例如黑屏），那么自己就刷不好了。
　　2.普通数据线是没办法刷机器的，只能传图片以及 MP3 等。
　　3.刷机时一定要确保手机电池电量在一半以上，这个很重要哦。
　　4、刷前请仔细阅读刷机操作说明
　　5、刷机时四个文件（或三个文件）的位置不要放错
　　6、建议在自己手机原版的基础上刷串号不变的DIY版本（如果该版本会改变串号，一般人还是不要刷的好，否则出了这样那样的问题自己难以解决)
　　7、不是任何手机都可以刷机的。比如NOKIA刷机需要专用的刷机盒，所以不能自己刷机的。所以只能去客服和一些手机商去刷机
　　8、不是任何问题都可以通过刷机解决的。有些问题可能是硬件问题，所以不要太依赖刷机
　　9、刷机最好在风险可控前提下的刷机。目前DIY的版本都是基于原版的，只不过是将原来的图片替换成另外的图片，将原来的铃声替换成另外的铃声，没有动核心部分。只是替换更改了部分图片、铃声或者菜单字符等，所以不应该有不良影响～只要做好自己手机软件的备份，按步骤操作，刷机是基本上没有风险的（大不了再刷回备份）。
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
注：以上内容来自“百度百科”。
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
OK，相对上面说的诺基亚、塞班，Android刷机就显得容易很多，不过你一定要记住：刷机（清除一切手机使用资料并更新操作系统，不会删除外置SD文件）不等于系统升级（在不会清除手机大部分使用资料的情况下更新操作系统，也不会删除外置SD文件）！！
========================================================================================
接下来切入正题——如何刷机。
刷机其实并不难，作者其实也刷过好多次了……一般情况下，刷机方法是没有区别的。
刷机方法：*首先，先备份你手机里你认为重要的程序、电话本条目、通话记录之类的，当然，备份在电脑端最好，本人一般是用“91手机助手”。
*接下来，把刷机包（zip格式的压缩文件）放在SD卡的根目录下（不明白的话就自己百度！），名称必须为update.zip（“zip”为文件后缀名，不知道的也是去百度），如要放置多个刷机包（……闲的啊……）请将在文件名（update）后依次加上2、3、4（如 第一个叫update.zip，第二个叫update2.zip，第三个叫update3.zip，以此类推）……若放置了多个刷机包，刷机前要选择（注：选择刷机包的功能不是全部Android手机/平板电脑都有的功能，一切请根据实际情况而定）。
*第三部开始正式刷机，刷机前请确定手机电量充足（一般在50%以上为最好），然后关机（如果本来就没开机就忽略此步骤……），按住“Home”键（此键不固定，有的手机没有正常“Home”键而换成“音量+”，请根据手机的实际情况选择相应的键）后再开机，之后你就可以根据提示进行下一步操作了！……

开始加入后面的内容（注：本方法只能对一部分Android手机适用）

操作须知：
1. 若成功进入升级模式（Recovery Mode）会在屏幕上看到一个带感叹号的三角形出现
若不成功 请拔掉电池 重试
2. 首页键（Home） ‐‐> 启动升级菜单
3. 侧键（Side）和光电鼠 ‐‐> 上下选择菜单
4. 菜单键（Menu）‐‐> 执行所选操作
在进行升级操作之前请确定：
1. 手机处于关机状态
2. SD卡根目录下已有升级包 update.zip
3. 电池电量较充足 （若升级过程中掉电可能会出问题）
*******************************************************************************
4. 请确保按键Home和Menu是可以使用的 否则进入升级模式后将再也无法退出
只能用PC端的烧写工具格式化（Format）手机
*******************************************************************************
操作步骤如下：

1. 手机关机

2. 按住“侧键 ‐ 上”
        再去按“power键”开机
3. 进入升级模式

4. 在进入升级模式后
        按“Home键”
        启动升级菜单

可以看到屏幕上方有2行
“Android system recovery <2e>” 和 “Android system recovery utility”字样
a. reboot system now ‐‐> 重启以退出升级模式
b. apply sdcard : update.zip ‐‐> 安装升级包update.zip
c. wipe data/factory reset ‐‐> 清除用户数据/恢复出厂设置
（顺带清除cache分区，所以一般d项可以不用）
d. wipe cache patition ‐‐> 清除cache分区
5. 按“Side‐down键”将蓝色高亮条移动到操作项c. wipe data/factory reset
        按“Menu键” 进入操作菜单
        按“Side‐down键”将蓝色高亮条移动到操作项 Yes – delete all user data
        再按“Menu键”
        若提示完成（completed） 则说明操作成功
        6. 按“Side‐down键”将蓝色高亮条移动到操作项b. apply sdcard: update.zip
再按“Menu键”
        若提示完成（completed） 则说明操作成功
7. 按“Menu键”完成操作项a. reboot system now
友情提示：
手机重启后即能正常开机
初次开机花费时间比较久，需要4—6分钟，请耐心等待；
