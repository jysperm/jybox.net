C++11新特征：auto类型推断、初始化列表、foreach循环


如题的三个特征，可以用一个很短的小程序来演示：

#include <iostream>
#include <vector>
int main()
{
    std::vector<int> array={1,2,3,4,5,6};
    for(auto i : array)
    {
        std::cout<<i<<std::endl;
    }
}
