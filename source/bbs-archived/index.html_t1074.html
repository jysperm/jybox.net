代码用字符串，你坑爹不坑爹啊

给你个好主意：
第三个参数$code你直接要用户指定一个函数名好不好啊，效果如下：

require_once('php-func-overload.php');

//三个重载形式的实现
function hello1()
{
  echo "hello php-func-overload\n.";
}

function hello2($a)
{
  echo ($a+$a)."\n";
}

function hello3($a,$b)
{
  echo "$a. Hello $b.\n";
}

//注册重载函数
_('hello','','hello1');
_('hello','$a:integer','hello2');
_('hello','$a:integer,$b="ab":string','hello3');

//演示：
hello();    // 第一个函数原型
hello(2);   // 第二个函数原型
hello(12, 'World'); // 第三个函数原型
