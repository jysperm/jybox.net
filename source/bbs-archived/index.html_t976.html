说说我的编码风格

这是我现行，以及今后打算使用的

缩进
html/js/css，2个空格缩进（其实这个是跟whtsky改的），如果有条件的话，发布的时候改为一个tab缩进（省流量）
其他代码，4个空格缩进
文件末尾无论如何都留一个以上的空行

变量
一般表示数值的话，无特殊要求都用int
表示布尔（是/否）用bool，用true和false表示是/否
用0表示空指针，而不是NULL（原因是NULL不是C++的关键字，而且不同的类库在NULL的类型上不一样，虽然不影响我，但是觉得不舒服的样子，为毛C++没有一个null的关键字呢？）
没有特殊情况，变量等到使用的时候再声明（听说C89不允许这样子？）

命名
变量/函数第一个单词小写，之后每个单词首字母大写
bool变量和返回bool的函数加is前缀
类名/命名空间名每个单词首字母大写
循环计数器依次用i j k l
类成员变量没有m_前缀（好像这个是mfc的风格）

空白：
所有能不用多余空格的地方都不加空格（&&  || 的左右是个例外）
大括号单独成行
复制代码
  1. if(1+1==2)
    {
        //.....
    }

每个函数中间隔一个空行
指针符（*），引用符（&）贴近变量/函数名一侧
复制代码
  1. inline OMessage &OMessage::append(QByteArray *data)


注释一律用双斜杠来注释，不用/*..*/
类
成员的顺序：public（slots）、protected（slots）、signals、private（slots）
类文件单独一个头文件/实现文件，文件名跟随类名（大小写混合），其他文件全小写
inline函数写在头文件里
